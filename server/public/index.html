Unix Time Stamp and Human readable date

This is an api to get Unix Time Stamp from readable date format and vice-versa.

Example:-


http://localhost:3000/12324324212434=>To get natural date from unix time stamp


http://localhost:3000/Sep%2019,%201994=>To get unix time from natural date


Please use following detail using natural date:-

  • format:- Month DD, YEAR
  • Month format:- Jan,Feb,March,April,May,June,July,August,Sep,Oct,Nov,Dec
  • No need to write %20, it is automatic
